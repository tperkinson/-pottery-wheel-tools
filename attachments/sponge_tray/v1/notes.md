# Sponge Tray Notes

## Files
- `sponge_tray_v1.scad`: curved tray + clamp sweep model.
- `sponge_tray_v1.stl`: prior exported mesh.

## Current Design
- Tray profile is integrated in 2D with the clamp profile and swept together.
- Tray dimensions and wall heights are parameterized in inch-based values.
- Uses core clamp profile from `../../../core_mount_system/c_clamp/v1/c_clamp_profile_topref_v1.scad`.
