# Phone Mount Notes

## Files
- `phone_mount_v1.scad`: parametric phone holder on C-clamp platform.
- `phone_mount_v1.stl`: last exported mesh.

## Current Design
- Clamp + pad are swept.
- Holder body is linear (not swept).
- Includes optional tilt brace for tilted holder support.
- Includes `render_holder_only` mode for fast fit-test prints.

## Quick-Tune Inputs
- `holder_tilt_deg`
- `holder_rest_radial_offset`
- `holder_width`
- `holder_shelf_depth`
- `holder_stop_height`
